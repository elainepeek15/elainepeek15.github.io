My CSS Typography Cheat Sheet

CSS Elements

Property Example Value Description
font-family Robato Specify the typeface in a ruleset
font-style italic Changing this selection will determine if the font is vertical or has a degree of slant
font-size 16px Sets text size. Font size values can be in many different units like absolute lengths and relative lengths.
text-align left Sets the text alignment of inline contents
font-weight bold Sets the text to be thinner or thicker. Can take value such as "lighter" or numerical value from 100 to 900.
text-decoration underline Adds lines to the text (underline, overline, strikethrough)
